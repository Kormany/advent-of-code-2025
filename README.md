# Advent of Code 2025 🧩

[![Java](https://img.shields.io/badge/Language-Java-blue.svg)](https://www.java.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

This repository contains my **Java solutions** for [Advent of Code 2025](https://adventofcode.com/2025).

---

## 📂 Project Structure

```
advent-of-code-2025/
├── Main.java
├── solutions/
│   ├── Day01.java
│   ├── Day02.java
│   └── ...
├── inputs/
│   ├── day01.txt
│   ├── day02.txt
│   └── ...
├── .gitignore
└── README.md
```

* `solutions/DayXX.java` – solution code for each day
* `inputs/dayXX.txt` – puzzle input for each day
* `Main.java` – central program to run all solutions

---

## 🚀 Usage

1. **Compile** all Java files:

```bash
javac Main.java solutions/*.java
```

2. **Run** the program:

```bash
java Main
```

> Make sure the `inputs/` folder is in the project root with the correct input files.
